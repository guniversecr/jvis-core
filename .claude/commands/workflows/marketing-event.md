## Event Type Variations

### Webinar (Virtual)

**Focus Areas:**
- Landing page optimization
- Email reminder sequence (day before, 1 hour, starting now)
- Live engagement (polls, Q&A)
- Recording follow-up for no-shows
- Shorter nurture cycle

### Conference (Virtual/Hybrid)

**Additional Considerations:**
- Multi-track agenda management
- Sponsor activation
- Networking facilitation
- Session-level tracking
- Longer planning timeline

### Trade Show

**Focus Areas:**
- Booth design and staffing
- Pre-show appointment setting
- Lead capture process
- On-site activation
- Post-show rapid follow-up

## Begin

Start by asking:

1. "What type of event is it? (webinar, conference, trade show, workshop)"
2. "What is the main objective? (leads, brand, education, sales)"
3. "What is the target date?"
4. "What is the target number of registrations/attendees?"

Then proceed to Phase 1: Event Planning.
